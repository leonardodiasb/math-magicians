# Math-Magicians

> A simple Calculator web app made with React. 

## Built With

- HTML/CSS
- Javascript
- React
- NPM

## Getting Started

### Install

- `git clone https://github.com/leonardodiasb/math-magicians.git`
- `cd math-magicians`
- `npm install`
- `npm run start`
